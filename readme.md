# CORBA IDL to C++ Compiler

This is a part of the Nirvana project.

## Purpose

This repository contains source code for Nirvana CORBA IDL to C++ compiler.

## Prerequisites

* CMake
* vcpkg
* flex
* bison

## How to build

Use CMake.

## Dependencies

IDL compiler front-end library: https://github.com/nirvanaos/idlfe
